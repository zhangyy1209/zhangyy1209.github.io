Personal Information

I am Yuanyang Zhang, a computer science Ph.D. student in University of California, Santa Barbara. My advisor is Prof. Linda Petzold. I received my bachelor degree at Beihang University in 2010.

My primary interests are Data Mining, Machine Learning, Statistics and Bioinformatics, especially with clinical data, time-series data and survival analysis. I am interested in building probabilistic models to solve these problems.

Meanwhile, I am interested in big data, especially with distributed machine learning algorithms and deep learning.

I am also interested in scalable web application design and building some cool services.

Research Projects

  • Mortality prediction in trauma clinical data using survival analysis, Oct. 2014 - present

    • Designed a cure model to make predictions of the mortality of trauma patients by using their static data.

  • Blood states identification in trauma clinical data using hidden Markov model, Mar. 2014 - Oct. 2014

    • Applied hidden Markov model to identify the blood states in trauma patients by using their temporal data.

  • Mortality prediction in trauma clinical data using temporal logistic regression model with elastic net, Sep. 2013 - Feb. 2014

    • Designed a temporal logistic regression model with elastic net to predict the mortality of trauma patients using both static and temporal data, which have a large amount of missing values.

  • Gene/probe set enrichment analysis on PTSD gene expression, methylation and mRNA data, Mar. 2013 - Sep. 2013

    • Modified the gene set enrichment analysis algorithm and applied on PTSD gene expression, methylation and mRNA data to obtain a list of significant GO terms and pathways.

  • Linear model to identify the impact of interventions to measurements using trauma clinical data, July. 2012 - Feb. 2013

    • Applied linear model to identify the impact of each intervention, such as red blood cells and plasma, to the measurements, such as protein C and pH value, by using trauma clinical data.

  • State discovery in trauma patients by identifying frequent occurring combination of discrete measurements, Jan. 2012 - July. 2012
  • Analysis of partition in scale-free network, Mar. 2011 - Sep. 2011

    • Derived the lower and upper bound of the expectation of cut when partitioning a scale-free graph into two parts, by using the preferential attachment model.

  • Accurate outdoor access point location using smart phones, Sep. 2010 - Mar. 2011
  • MAC protocol design for future wireless system, Sep. 2009 - Feb. 2010
